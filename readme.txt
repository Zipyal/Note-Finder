Программа включает три функции:

    getNoteName — определяет имя ноты.
    getOctave — вычисляет номер октавы.
    getNoteAndOctave — объединяет логику вычисления ноты и октавы.

Имеется цикл в функции main, позволяющий пользователю многократно вводить позиции нот. Для завершения программы нужно ввести -1.




Цель программы: Определение ноты и её октавы по местоположению на нотном стане в скрипичном ключе (ключ "соль").

    Для каждой заданной линии (или промежутка) пользователь получает название ноты и номер октавы.

Пользовательский ввод:

    Пользователь вводит местоположение ноты в числовой форме, где каждая линия и промежуток имеют определённый номер (например, 2 для G первой октавы).
    Программа позволяет вводить значения в диапазоне от 0 до 10.

Вывод результата:

    Программа выводит название ноты (например, G, A, B) и её октаву.

Дополнительные возможности:

    Включён цикл, чтобы пользователь мог вводить несколько значений за один сеанс.
    Условие выхода: ввод -1.
